# vdpSchool School Management CRUD Application

A simple full-stack CRUD web application built to demonstrate object-oriented design principles, RESTful API development, and frontendâ€“backend integration using Angular and C# (.NET).

This project models a basic school domain, allowing users to manage entities such as students, teachers, and classes through a web interface.

The application consists of:

- Angular frontend for user interaction
- C# ASP.NET Web API backend for business logic and data handling
- A simple domain model designed using object-oriented principles

The goal of this project is to demonstrate:

- Clean separation of concerns
- Basic CRUD operations
- Practical application of OOP concepts
- Full-stack communication via HTTP APIs
